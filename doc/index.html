ZAWS - Amazon Web Service access from Common Lisp

ZAWS is a Common Lisp library for creating Amazon Web Service client libraries. It is available under a BSD-style license; see LICENSE.txt for details. The latest version is 1.0, released on Febrary 2nd, 2012.

Download shortcut: http://www.xach.com/lisp/zaws.tgz
